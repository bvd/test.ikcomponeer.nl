This website emulates the behavior of a portal that uses the composition environment by embedding it.

Step 1 - obtain a client key

You can obtain a client key via info@adeptive.nl. The client key identifies your platform embedding the composition environment.

Step 2 - identify your user.

Your user will need to have a unique identification that will be used with the composition environment to store and retrieve his material.

Step 3 - choose an activity for the user

Based on a number of template activities, you may embed the specific set of music and other behavior of the composition environment.

Step 4 - generate an access token for the user

On your own server you must generate an access token for the user according to the example project here.
